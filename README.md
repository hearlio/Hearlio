## 💕 What is Hearlio?

Hearlio is an AI companion that helps people navigate their relationships, friendships, family issues, and workplace stress. It listens, understands, and gives warm, personalized advice — like talking to a caring best friend who never judges you.

---

## ✨ Features

- 💬 **AI-Powered Chat** — Real, warm, empathetic responses powered by Claude AI
- 🧠 **Memory System** — Remembers your story across sessions, no need to repeat yourself
- 📂 **Heart Stories** — Save multiple ongoing conversations like projects
- 🎙️ **Voice Input** — Speak instead of type (works in Chrome)
- 🔊 **Voice Output** — Hearlio speaks back in male or female voice
- 🌍 **6 Languages** — English, Urdu, Arabic, German, French, Spanish
- 🔄 **Auto RTL** — Arabic and Urdu text automatically right-to-left
- 😊 **Mood Check-in** — Sets the tone before every conversation
- 🎨 **Multi-color Chat** — Beautiful colorful chat bubbles
- 📱 **Mobile Friendly** — Works on all screen sizes

## 🛠️ Setup Instructions

### Step 1 — Get your Anthropic API Key
1. Go to [console.anthropic.com](https://console.anthropic.com)
2. Sign up and add credits (minimum $5)
3. Go to **API Keys** → **Create Key** → Copy it

### Step 2 — Add your API Key
1. Open `index.html` in Notepad
2. Find this line:
```
const API_KEY = "YOUR_ANTHROPIC_API_KEY_HERE";
```
3. Replace it with your real key:
```
const API_KEY = "sk-ant-xxxxxxxxxxxxxxx";
```
4. Save the file

### Step 3 — Deploy for Free (GitHub Pages)
1. Create a free account at [github.com](https://github.com)
2. Create a new repository named `hearlio` → set to **Public**
3. Upload your `index.html` file
4. Go to **Settings** → **Pages** → Source → select **main** → Save
5. Your live link: `https://yourusername.github.io/hearlio`
